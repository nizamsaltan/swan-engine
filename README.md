# Swan Engine
Swan Engine is 3D graphics engine written in C++ using OpenGL. This engine specifically designed for AR devices and apps.

### Requirements
sudo pacman -S assimp glfw *cmake* 


### Tools
- Glm -> Math Library
- ImGUI -> GUI
- Assimp -> Model Importer
- STB -> Image Loading
- GLFW -> Window
